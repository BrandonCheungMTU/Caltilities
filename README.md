Caltilities is a powerful, open-source calculator application designed for complex mathematical computations. Built with Python and QML, it combines robust symbolic math capabilities with a modern, intuitive user interface.

## ✨ Features

- **Symbolic Mathematics**: Solve equations, differentiate, integrate using SymPy
- **Graphing**: 2D and 3D function plotting with Matplotlib
- **Unit Conversion**: Comprehensive unit conversions
- **Matrix Operations**: Linear algebra operations
- **Custom Functions**: Define and save custom functions
- **Modern UI**: Smooth animations, dark/light themes, keyboard shortcuts

## 📋 Prerequisites

- Python 3.9 or higher
- PySide6
- SymPy
- NumPy
- Matplotlib
